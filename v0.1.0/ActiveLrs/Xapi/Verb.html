Overview

Represents an xAPI Verb object.

A Verb describes the action that the Actor performed on the Object. Each Verb must have an â€˜id` (IRI) and may optionally include a human-readable `display` map of language codes to strings.

def initialize(attributes = {})
  self.id = attributes["id"] if attributes["id"]

  if attributes["display"]
    self.display = attributes["display"]
    locale = ActiveLrs.configuration.default_locale
    self.name = attributes["display"][locale] || attributes["display"].values.first
  end
end

def to_h
  node = {}
  node["id"] = id.to_s if id
  node["display"] = display if display
  node
end
